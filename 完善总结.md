## 使用方法

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/13888285815/qoder-chinese-copyright-application-skill.git
cd qoder-chinese-copyright-application-skill

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行脚本
python3 scripts/generate_copyright_docs.py /path/to/your/project

# 4. 转换PDF
python3 scripts/convert_to_pdf.py copyright-application-materials/
```

### 工作流程

1. **启动脚本** → 自动分析项目结构
2. **输入著作权人信息** → 交互式收集
3. **输入软件信息** → 开发目的、功能特点等
4. **生成文档** → 自动生成4个文档
5. **转换为PDF** → 可选，转换为PDF格式
