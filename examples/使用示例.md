# 使用示例

本文档提供了 qoder-chinese-copyright-application-skill 的详细使用示例。

## 示例1：生成微信小程序著作权申请材料

假设您有一个微信小程序项目位于 `/Users/developer/my-weapp-project`

### 步骤1：安装依赖

```bash
cd qoder-chinese-copyright-application-skill
pip install -r requirements.txt

# 安装wkhtmltopdf（用于PDF转换）
# macOS
brew install wkhtmltopdf
# Ubuntu/Debian
sudo apt-get install wkhtmltopdf
```

### 步骤2：运行生成脚本

```bash
python3 scripts/generate_copyright_docs.py /Users/developer/my-weapp-project
```

### 步骤3：交互式输入信息

脚本会提示您输入以下信息：

```
============================================================
软件著作权申请材料生成工具
============================================================

在开始生成文档之前，请提供著作权人信息：

1. 著作权人名称（个人姓名或公司全称）: 张三
2. 证件类型（身份证/营业执照）: 身份证
3. 证件号码: 110101199001011234
4. 联系地址: 北京市朝阳区xxx街道xxx号
5. 邮政编码: 100000
6. 联系人姓名: 张三
7. 联系电话: 13800138000
8. 电子邮箱: zhangsan@example.com

请提供软件信息（可以直接回车使用默认值）：

1. 开发目的（限50字）: 提供便捷的社区服务，方便居民生活
2. 面向领域/行业（限50字）: 社区服务
3. 软件的主要功能（限200字）: 提供社区公告、物业缴费、报修服务等功能
4. 软件的技术特点（限100字）: 基于微信小程序平台，快速开发和迭代
5. 软件类型（APP/小程序/游戏软件等）: 小程序
```

### 步骤4：查看生成结果

文档将生成在 `/Users/developer/my-weapp-project/copyright-application-materials/` 目录下：

```
copyright-application-materials/
├── 软件著作权登记申请表.md
├── 源代码文档.md
├── 用户手册.md
└── 设计说明书.md
```

### 步骤5：转换为PDF

```bash
python3 scripts/convert_to_pdf.py copyright-application-materials/
```

## 示例2：指定输出目录

## 示例3：批量转换PDF

将生成的所有Markdown文档批量转换为PDF：

```bash
python3 scripts/convert_to_pdf.py copyright-application-materials/
```

转换单个文件：

```bash
python3 scripts/convert_to_pdf.py \
  copyright-application-materials/软件著作权登记申请表.md \
  copyright-application-materials/申请表.pdf
```

## 示例4：使用在线工具转换PDF

如果您不想安装wkhtmltopdf，可以使用以下在线工具：

### 4.1 Typora

1. 下载安装 Typora：https://typora.io/
2. 打开生成的 .md 文件
3. 选择菜单 "文件" → "导出" → "PDF"
4. 保存即可

### 4.2 使用Pandoc

```bash
# 安装pandoc
# macOS
brew install pandoc
# Ubuntu/Debian
sudo apt-get install pandoc

# 转换单个文件
pandoc 软件著作权登记申请表.md -o 软件著作权登记申请表.pdf

# 批量转换
for file in *.md; do
  pandoc "$file" -o "${file%.md}.pdf"
done
```

### 4.3 GitHub + 导出

1. 将 .md 文件推送到 GitHub
2. 打开文件
3. 点击 "Raw" 按钮查看原始内容
4. 使用浏览器打印功能导出为PDF

## 示例5：常见问题排查

### 问题1：Python脚本找不到模块

**错误信息：**
```
ModuleNotFoundError: No module named 'markdown'
```

**解决方案：**
```bash
pip install markdown pdfkit
```

### 问题2：PDF转换失败

**错误信息：**
```
OSError: wkhtmltopdf exited with 1
```

**解决方案：**
1. 确保已安装 wkhtmltopdf
2. macOS用户：如果提示安全警告，运行：
   ```bash
   sudo xattr -r -d com.apple.quarantine /usr/local/bin/wkhtmltopdf
   ```

### 问题3：中文显示问题

**解决方案：**
1. 确保系统安装了中文字体
2. 在 convert_to_pdf.py 中修改字体设置：
   ```python
   font-family: "Microsoft YaHei", "SimHei", sans-serif;
   ```

## 输出文件检查清单

生成文档后，请检查以下内容：

- [ ] 所有文档开头都包含著作权人名称
- [ ] 申请表中所有必填字段都已填写
- [ ] 源代码文档包含前后各30页
- [ ] 所有文档中的软件名称、版本号一致
- [ ] PDF文件已成功生成

## 提交申请前检查

在提交软件著作权申请前，请确保：

1. 所有信息真实准确
2. 源代码为原创，未侵犯他人著作权
3. 申请材料完整（申请表、源代码、用户手册、设计说明书）
4. PDF文件可正常打开和打印
5. 联系方式正确，以便版权中心联系
